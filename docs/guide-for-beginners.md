# 小红书自动发布 Agent - 新手指南

> 📖 本指南专为**没有技术背景**的普通用户设计，从零开始教你使用工具。

---

## 🎯 这个工具能帮你做什么？

✅ **3 分钟发布一篇小红书笔记**
- 自动写标题（不超过 20 字，符合平台规范）
- 自动写正文（带 emoji 和话题标签）
- 自动配图片（AI 生成或兜底封面图）
- 自动发布到小红书

✅ **7 种内容风格**
- 清新自然、专业干货、生活分享、种草推荐等

✅ **完全可视化操作**
- 不需要写代码
- 不需要记命令
- 点点鼠标就能用

## 🚀 快速开始（5 分钟配置，终身使用）

### 第 1 步：安装（2 分钟）

**如果你已经安装了 Node.js：**

```bash
# 1. 下载项目
git clone https://github.com/jingjiansoft/xiaohongshu-agent.git
cd xiaohongshu-agent

# 2. 安装依赖
npm install
```

**如果你没有 Node.js：**

1. 下载安装 Node.js：https://nodejs.org/（选择 LTS 版本）
2. 安装完成后，重复上面的步骤

### 第 2 步：启动（10 秒）

```bash
# 一键启动
./start.sh
```

启动后会看到：
```
🚀 启动小红书自动发布 Agent...
📡 启动后端 API 服务...
   后端服务运行在 http://localhost:3001
🌐 启动 Web 应用...
   Web 应用运行在 http://localhost:3000
✅ 所有服务已启动
```

### 第 3 步：配置模型（2 分钟）

1. **打开配置页面**
   
   浏览器访问：http://localhost:3000/settings

2. **点击"模型配置"标签页**

3. **选择文本模型**
   
   推荐选择：**通义千问** 或 **DeepSeek**（性价比高）

4. **填入 API Key**
   
   - 通义千问：https://dashscope.console.aliyun.com/
   - DeepSeek：https://platform.deepseek.com/
   - 其他模型提供商官网获取

5. **选择图片模型**
   
   推荐选择：**通义万相**（和通义千问同一个 API Key）

6. **点击"保存配置"**
   
   看到"配置保存成功"提示即可

### 第 4 步：登录小红书（2 分钟）

1. **打开登录页面**
   
   浏览器访问：http://localhost:3000/login

2. **点击"打开小红书登录页面"**
   
   会自动打开新标签页

3. **扫码登录**
   
   用小红书 APP 扫码

4. **返回检查状态**
   
   看到"已登录"绿色提示即可

## 📝 开始创作（3 分钟一篇）

### 方法 1：Web 界面（推荐）

#### 1. 打开首页

浏览器访问：http://localhost:3000

#### 2. 输入主题

在"主题"输入框填写：
```
例如：健康早餐、旅行攻略、好物推荐...
```

#### 3. 选择风格

下拉选择：
- 🌸 清新自然
- 📚 专业干货
- 🏠 生活分享
- 🛍️ 种草推荐
- 💭 情感共鸣

#### 4. （可选）填写关键词

用逗号分隔：
```
例如：营养, 简单, 快手早餐
```

#### 5. 点击"生成内容"

等待 30 秒 -2 分钟，AI 会自动生成：
- ✅ 标题（20 字以内）
- ✅ 正文（带 emoji）
- ✅ 话题标签（3-5 个）
- ✅ 配图（1-3 张）

#### 6. 预览和编辑

查看生成的内容，可以：
- 修改标题
- 修改正文
- 增删话题
- 重新生成图片

#### 7. 点击"发布到小红书"

自动完成：
- 🖼️ 上传图片
- ✍️ 填写标题和正文
- 🏷️ 添加话题标签
- ✅ 点击发布按钮

#### 8. 查看发布结果

发布成功后会显示：
- ✅ 发布成功
- 🔗 笔记链接（可点击访问）

### 方法 2：命令行（适合批量）

```bash
# 生成并发布
npm run publish:content -- -t "健康早餐" -s "生活分享"

# 只生成不发布
npm run publish:content -- -t "健康早餐" -s "生活分享" --no-publish
```

### 1. 写出好标题的技巧

- 在主题中体现关键词

**添加详细关键词：**

## ❓ 常见问题

### Q1: API Key 从哪里获取？

**通义千问/通义万相：**
1. 访问：https://dashscope.console.aliyun.com/
2. 注册/登录阿里云账号
3. 开通"通义千问"和"通义万相"服务
4. 创建 API Key
5. 复制到配置页面

**DeepSeek：**
1. 访问：https://platform.deepseek.com/
2. 注册/登录
3. 在"API Keys"页面创建
4. 复制到配置页面

### Q2: 登录状态失效怎么办？

1. 访问：http://localhost:3000/login
2. 点击"刷新状态"
3. 如果显示"未登录"，重新扫码登录即可

### Q3: 发布失败怎么办？

**检查登录状态：**
- 访问登录页面检查是否已登录

**检查模型配置：**
- 访问设置页面检查 API Key 是否配置

**查看详细错误：**
- 页面会显示具体错误信息
- 截图发到 GitHub Issues 求助

### Q4: 图片生成很慢或失败？

**正常现象：**
- AI 图片生成需要 30 秒左右
- 失败会自动使用兜底图片

**优化建议：**
- 检查网络连接
- 检查图片模型 API Key 是否正确
- 使用通义万相（国内速度快）

### Q5: 标题能不能超过 20 字？

**不能，也不建议：**
- 小红书标题限制 20 字
- 超过会被平台截断
- 系统会自动截断并添加"..."

### Q6: 一次能发几张图片？

**最多 9 张：**
- 小红书支持 1-9 张图片
- 建议 3-6 张效果最好
- 可以在生成时设置图片数量

### Q7: 支持视频发布吗？

**开发中：**
- 目前仅支持图文发布
- 视频发布功能预计 1 个月内上线

### Q8: 可以在手机上用吗？

**可以：**
- Web 界面支持手机浏览器访问
- 但推荐使用电脑（操作更方便）
- 移动端优化开发中

## 🛠️ 故障排除

### 问题：启动失败

**错误：`npm: command not found`**
- 解决：安装 Node.js（https://nodejs.org/）

**错误：`Cannot find module`**
- 解决：运行 `npm install` 安装依赖

### 问题：无法访问页面

**错误：`无法访问此网站`**
- 解决：检查服务是否启动
- 运行 `./start.sh` 重新启动

### 问题：生成内容为空

**可能原因：**
- API Key 未配置或错误
- 网络连接问题
- 模型服务不可用

**解决方法：**
1. 检查模型配置
2. 测试 API Key（设置页面有"测试配置"按钮）
3. 更换模型提供商

### 问题：发布中断

**可能原因：**
- 登录态过期
- 网络不稳定
- 小红书页面结构变化

**解决方法：**
1. 重新登录小红书
2. 检查网络连接
3. 到 GitHub Issues 反馈
